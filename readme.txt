ANTON FT - Daily Time Record System v2.0
DepEd Public Schools
===========================================

WHAT IS ANTON FT?
------------------
ANTON FT (Automated Notification and Timekeeping Online Network Faculty Tracker) is a FREE Daily Time Record system designed specifically for Philippine public schools following DepEd standards.

FEATURES:
---------
✓ Selfie-based login
✓ Geotagged location tracking
✓ Offline capability (works without internet)
✓ Automatic sync when online
✓ DepEd-standard DTR format
✓ Export to Excel
✓ School admin management
✓ Superadmin controls

HOW TO INSTALL:
---------------

Option 1: WEB BROWSER (Easiest)
1. Open any web browser (Chrome, Firefox, Edge)
2. Go to: [YOUR HOSTED URL]
3. Click "Install App" button in the header
4. The app will install like a native application

Option 2: OFFLINE (USB/File)
1. Extract the ANTON_FT folder to your computer
2. Double-click index.html
3. The app opens in your default browser
4. Bookmark the page for easy access

Option 3: ANDROID APP
1. Open Chrome and go to the app URL
2. Tap the "Install App" banner at the bottom
3. The app installs like a regular Android app

Option 4: DESKTOP APP (Windows/Mac)
1. Install Chrome or Edge browser
2. Go to the app URL
3. Click the install icon in the address bar
4. The app opens as a standalone window

FIRST TIME USE:
--------------
1. Click "Login via Selfie"
2. The system will assign a sample user
3. You can change the school details in the admin section
4. Start logging time and events

FOR SCHOOL ADMINS:
-----------------
1. Login as "Admin Dela Cruz" (click selfie button multiple times)
2. Update your school name, address, and location
3. The admin section appears for configuration
4. Teachers can now use the updated school info

FOR SUPERADMINS:
---------------
1. Login as "Superadmin Tan"
2. Full access to all features
3. Can assign school admins (feature coming soon)
